# Linux进程间通讯信号

---

---

| 信号 | 名称 |              描述              |
| :--: | :--: | :----------------------------: |
|  1   | HUP  |              挂起              |
|  2   | INT  |              中断              |
|  3   | QUIT |            结束运行            |
|  9   | KILL |           无条件终止           |
|  11  | SEGV |             段错误             |
|  15  | TERM |           尽可能终止           |
|  17  | STOP |    无条件停止运行，但不终止    |
|  18  | TSTP |  停止或暂停，但继续在后台运行  |
|  19  | CONT | 在**STOP**或**TSTP**后恢复运行 |
