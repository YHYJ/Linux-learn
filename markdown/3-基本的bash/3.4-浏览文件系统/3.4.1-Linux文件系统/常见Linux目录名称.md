常见的目录名称基于**文件系统层级标准(filesystem hierarchy standard，FHS)**

[FHS主页](http://www.pathname.com/fhs)

| 目录   | 用途                                                    |
| ------ | ------------------------------------------------------- |
| /      | 虚拟目录(virtual diretory)的根目录                      |
| /bin   | 二进制目录，存放许多用户级的GNU工具                     |
| /boot  | 启动目录，存放启动文件                                  |
| /dev   | 设备目录，Linux在此创建设备节点                         |
| /etc   | 系统级配置文件目录                                      |
| /home  | 主目录，Linux在此创建用户目录                           |
| /ib    | 库目录，存放系统和应用程序的库文件                      |
| /media | 媒体目录，可移动媒体设备的常用挂载点                    |
| /mnt   | 挂载目录，另一个可移动媒体设备的常用挂载点              |
| /opt   | 可选目录，常用于存放第三方软件包课数据文件              |
| /proc  | 进程目录，存放现有硬件和当前进程的相关信息              |
| /root  | root用户的主目录                                        |
| /sbin  | 系统二进制目录，存放许多GNU管理员级工具                 |
| /run   | 运行目录，存放系统运行时数据                            |
| /srv   | 服务目录，存放本地服务的相关文件                        |
| /sys   | 系统目录，存放系统硬件信息的相关文件                    |
| /tmp   | 临时目录，可以在该目录中创建和删除临时工作文件          |
| /usr   | 用户级二进制目录，大量用户级的GNU工具和数据文件存放于此 |
| /var   | 可变目录，存放经常变化的文件，比如日志                  |
