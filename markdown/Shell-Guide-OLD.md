## 基础

### 传参

> Shell语言内置了用于接收参数的变量，变量之间以空格分隔

| 变量           | 含义                       | 备注 |
| -------------- | -------------------------- | ---- |
| $0             | 脚本文件名                 |      |
| $#             | 总共的参数数量             |      |
| $*             | 所有位置的参数值           |      |
| $?             | 上一条命令的执行结果返回值 |      |
| \$1...\$3...$n | 第N个位置的参数值          |      |

## 进阶

### 条件测试语句

> **测试语句格式**：`[ 条件表达式 ]`
>
> 4种条件测试语句（按照测试对象分）：

- 文件测试语句：使用指定条件判断**文件是否存在**或者**权限是否满足**等

  | 运算符 | 作用                               | 备注  |
  | ------ | ---------------------------------- | ----- |
  | -d     | 测试文件是否为目录类型             | dir   |
  | -e     | 测试文件是否存在                   | exit  |
  | -f     | 判断是否是普通文件                 | file  |
  | -r     | 测试当前用户是否对该文件有读取权限 | read  |
  | -w     | 测试当前用户是否对该文件有写入权限 | write |
  | -x     | 测试当前用户是否对该文件有执行权限 | exec  |

- 逻辑测试语句

  | 运算符 | 名称     | 作用                                   |
  | ------ | -------- | -------------------------------------- |
  | &&     | 逻辑“与” | 当前一个命令执行成功才会执行后一个命令 |
  | \|\|   | 逻辑“或” | 当前一个命令执行失败才会执行后一个命令 |
  | !      | 逻辑“非” | 把条件测试中的结果取反值               |

- 整数值比较语句

  | 运算符 | 名称     | 作用                  | 备注 |
  | ------ | -------- | --------------------- | ---- |
  | -eq    | 等于     | equal                 |      |
  | -ne    | 不等于   | not equal             |      |
  | -gt    | 大于     | greater than          |      |
  | -lt    | 小于     | less than             |      |
  | -le    | 小于等于 | less than or equal    |      |
  | -ge    | 大于等于 | greater than or equal |      |

- 字符串比较语句

  | 运算符 | 作用                   | 备注     |
  | ------ | ---------------------- | -------- |
  | =      | 比较字符串内容是否相同 | 默认相同 |
  | !=     | 比较字符串内容是否不同 | 默认不同 |
  | -z     | 判断字符串内容是否为空 | 默认为空 |
