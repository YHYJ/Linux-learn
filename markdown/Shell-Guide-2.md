## SHELL表达式

> 区分不同情况来执行不同操作

### 测试语句

> Shell环境根据命令执行后的返回状态值($?)来判断是否成功执行，返回值为0表示成功，其他值表示失败
>
> test命令可以对特定条件进行测试，并根据返回值判断条件是否成立（为0则成立）
>
> `test`命令和`[ ... ]`效果一样

* test 条件表达式

* [ 条件表达式 ]

  ```shell
  [ 1 = 2 ]	# 两侧空格
  ```

---

### 条件表达式

> 将测试语句的返回值作为条件判断是否执行某个操作

#### 逻辑判断

> 一般用于判断多个条件之间的依赖关系

* &&

  ```shell
  # 命令1 && 命令2
  # 成则都成，败则都败，根据命令1的执行状态返回值
  [ 1 = 1 ] && echo "命令1条件成立，命令2得以执行"
  [ 1 = 2 ] && echo "命令1条件不成立，命令2无法执行"
  ```

* ||

  ```shell
  # 命令1 || 命令2
  # 只能执行其一
  [ 1 = 1 ] || echo "命令1条件成立，命令2无法执行"
  [ 1 = 2 ] || echo "命令1条件不成立，命令2得以执行"
  ```

#### 文件判断

* `-f`

  ```shell
  # 判断输入的内容是否是一个文件
  [ -f ./test.sh ] && echo "是文件"
  [ -f ./ ] || echo "不是文件"
  ```

* `-d`

  ```shell
  # 判断输入的内容是否是一个目录
  [ -d ./ ] && echo "是目录"
  [ -d ./test.sh ] || echo "不是目录"
  ```

* `-x`

  ````shell
  # 判断输入的内容是否有执行权限
  [ -x ./test.sh ] && echo "可执行"
  [ -x ./ ] || echo "不可执行"
  ````

#### 数值判断

```shell
# 判断给定的两个数字的大小关系
[ 1 -eq 1 ] && echo "1=1"
[ 1 -eq 2 ] || echo "1!=2"
```

> num1 -eq num2	等于
>
> num1 -ne num2	不等
>
> num1 -gt num2	大于
>
> num1 -lt num2		小于

#### 字符判断

```shell
# 判断字符的内容是否一致
[ "1" == "1" ] && echo "一致"
[ "1" != "2" ] && echo "不一致"
```

> str1 == str2
>
> str1 != str2

### 计算表达式

> 对具体的内容进行算数计算

1. $(( 计算表达式 ))

   ```shell
   echo $(( 1+1 ))		# 2
   ```

   > 表达式范围内空格不受限制
   >
   > 只能进行加减乘除运算和()
   >
   > 只能做整数运算

2. let 计算表达式

   ```shell
   num=1
   let i=i+1
   echo $i		# 2
   ```

   > 表达式必须是一个整体，中间不能出现空格等特殊字符
